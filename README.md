# CurseForge Scripts

This is a collection of some scripts I use with CurseForge.com to make it easier to use for creators.

---

## Export Transactions to CSV

The `curseforge-transactions-export-csv.user.js` script adds a button to the Transactions page to download the whole transaction history to a CSV file.

## 10K Transaction Rows Per Page

The `curseforge-transactions-10k.user.js` script adds a new **10000** option to the **Rows per Page** dropdown of the Transactions page, to make it, well, load 10k entries as once.<br>
Use with caution. Creates some lag.
